# Trabajo - Sistema de videovigilancia con Raspberry Pi
---
El objetivo de este proyecto es simplemente diseñar e implementar un sistema que sea capaz de detectar movimientos y registrarlos únicamente cuando ocurran. De esta manera, se optimiza el almacenamiento y evitamos guardar información poco necesaria. La elección de este proyecto se debe a la gran utilidad práctica que tiene, asi como la posibilidad de trabajar con tecnologías nuevas y reales que combinan hardware y software en un sistema funcional y adaptable a las necesidades actuales.
